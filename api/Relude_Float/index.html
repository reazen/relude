let approximatelyEqual: tolerance:float => float => float => bool;

`approximatelyEqual(~tolerance=t, x, y)` returns `true` if `x` and `y` are within `t` of one another, `false` otherwise. The `~tolerance` must be non-negative.

### Example ```re approximatelyEqual(~tolerance=0.01, 1.500, 1.495) == true; approximatelyEqual(~tolerance=0.01, 1.500, 1.49) == false; ```

let toInt: float => int;

`toInt` converts a float to an int by dropping the fractional part.

Floats that can't be represented as an int (e.g. `infinity`, `nan`) will return 0.

let fromInt: int => float;

`fromInt` converts an int (e.g. 1) to its floating-point representation (1.0)

let fractionalPart: float => Ring.t;

Return only the decimal portion as a positive floating point number, with the whole-number portion set to 0.

Note that the returned value is subject to floating point rounding errors and probably won't be exactly equal to the fractional part of the original number.

### Example ```re fractionalPart(3.141592) ~= 0.141592; fractionalPart(-12.3456) ~= 0.3456; ```

let toPrecision: decimals:int => float => float;

`toPrecision` drops decimals so that the given float has no more than the requested number of decimals.

### Example ```re toPrecision(~decimals=2, 3.141592) == 3.14; toPrecision(~decimals=4, -4.99999999999) == -4.9999; ```

let show: float => string;

`toString(x)` returns the string representation of `x`. Note that this may not be in the same form as you wrote it in the source code.

### Example ```re toString(3.49) == "3.49"; toString(7.0000) == "7"; toString(1.0e+03) == "1000"; ```

let fromString: string => option(float);

`fromString(s)` returns `Some(x)` if the given string represents a valid floating point number, `None` otherwise.

### Example ```re fromString("1000.0") == Some(1000.0); fromString("-5.25E2") == Some(-525.0); fromString("3.4.5") == None; ```
